1. Build Code Component

LAB SCENARIO

Working as part of the PrioritZ fusion team you have been asked to create a Power Apps code component to allow drag and drop priority ranking of items in the PrioritZ Ask Power App. You will build a code component using the React JavaScript framework. The code component approach is used to address the requirement because there isnâ€™t a similar control already built-in.

You have collaborated with the app makers to identify the following properties to allow them to configure the code component in the app:

â€¢ BackgroundColor

â€¢ DragBackgroundColor

â€¢ ItemHeight

â€¢ FontSize

â€¢ FontColor

The PrioritZ Ask app will prepare a collection of the items to rank that will be bound as the dataset for the code component.
When an item is dragged and dropped the code component will raise an OnSelect event that will be handled by the hosting app.
The hosting app will update the collection items with their new rank. The code component will be stateless.

In Exercise 1 you will build the code component.

  1. Start Visual Studio Code.
  2. Select the Power Platform tab and make sure your Dev Auth profile is selected.

  1. Click Terminal and select New Terminal.

  1. Run the command below to switch to the PrioritZDnRanking directory you just created.

1.2 Review and run the code componentâ€‹

  1. In the terminal, run the command below to open the project in your current Visual Studio Code session.
code -a .

  1. Review the created code component files.

  2. Expand the PrioritZDnDRanking folder and then expand the component folder.

  3. Open the ControlManifest.Input.xml file. The manifest is the metadata file that defines the component including the properties exposed to the hosting app.

  4. In the ControlManifest.Input.xml file locate data-set XML element.

  5. Change the name to items and the display-name-key to items. This defines the property the app will bind to a collection of items.

  6. Add the following properties after the closing tag of the data-set element </data-set>.

<property name="BackgroundColor" display-name-key="Background color" usage="input" of-type="SingleLine.Text" default-value="#F3F2F1"/>
<property name="DragBackgroundColor" display-name-key="Drag background color" usage="input" of-type="SingleLine.Text" default-value="lightgreen"/>
<property name="ItemHeight" display-name-key="Item height" usage="input" of-type="Whole.None" default-value="32"/>
<property name="FontSize" display-name-key="Font size" usage="input" of-type="Whole.None" default-value="12"/>
<property name="FontColor" display-name-key="Font color" usage="input" of-type="SingleLine.Text" default-value="#333333"/>

  1. Locate <resources> and uncomment css resource. This will ensure that our styles will be bundled with the code component when it is deployed.

  2. Notice the following two resources. This declares the componentâ€™s dependency on these two libraries. This is a result of specifying â€“framework React on initialization.

<platform-library name="React" version="16.8.6" />
<platform-library name="Fluent" version="8.29.0" />

  1. Click File and select Save All.

  2. Make sure you still have the ControlManifest.Input.xml file selected and then click New Folder.

  3. Name the new folder css.

  4. Select the new css folder you created and then click New File

  5. Name the new file PrioritZDnDRanking.css.

  6. Paste the following css into the PrioritZDnDRanking.css file.
